# 2D Game Physics Engine Programming

This repository contains the teachings from Pikuma's [2d Game Engine Programming](https://pikuma.com/courses/game-physics-engine-programming) Course.

## Setup

In order to be able to run the project, you'll need the following packages installed:
- SDL 2, SDL 2 image, SDL 2 gfx
- CMake
- Optionals: clang, ninja

For Fedora 43 I use:
```bash
sudo dnf install clang cmake sdl2-compat-devel SDL2_image-devel SDL2_gfx-devel
```
